# Week 5 – Git & Angular Hands-On Exercises

## Overview
This folder contains hands-on exercise screenshots for Week 5 of the CTS Digital Nurture 5.0 Java Full Stack Engineer program, covering Git version control and Angular front-end development.

## Topics Covered
- **Git**
  - Git basics: init, add, commit, push, pull
  - Branching and merging
  - Resolving merge conflicts
  - Working with remote repositories

- **Angular**
  - Angular project setup and structure
  - Components, modules, and data binding
  - Services and dependency injection
  - Routing and navigation

This folder contains screenshots (output(1).png through output(N).png) documenting the completion of each hands-on exercise listed above.
